# Approximate-Nearest-Neighbor-Framework

# HOW TO RUN

bash compile.sh <br />
bash run.sh [input_file] [output_file] [Integer value of k] <br />

# DEMO RUN

bash compile.sh <br />
bash run.sh sample_inputs/sample4.in sample_outputs/sample4.out 5 <br />
